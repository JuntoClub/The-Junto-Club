# The Junto Club Fully Coded Website

This is a real responsive HTML/CSS site using the approved vintage newspaper/sketchbook aesthetic.

## Upload to GitHub

Upload these files to the root of your repo:
- index.html
- style.css
- README.md

There are no required image assets in this version.
